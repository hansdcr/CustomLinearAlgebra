本目录用于记录学习过程中提出的问题和详细解答。

## 📂 目录结构

```
docs/QA/
├── README.md                           # 本文件
├── iterXX_QN_问题简述.md               # 问题详解文档
└── QA索引.md                           # 所有问题的索引
```

---

## 📝 文件命名规范

### 命名格式

```
iter{迭代编号}_Q{问题编号}_{问题简述}.md
```

### 示例

- `iter01_Q1_向量归一化原理.md` - 迭代1的第1个问题
- `iter02_Q1_向量加法几何意义.md` - 迭代2的第1个问题
- `iter03_Q2_点积为什么等于投影长度.md` - 迭代3的第2个问题

### 命名规则

1. **迭代编号**: 两位数字,如 `01`, `02`, `15`
2. **问题编号**: 从1开始递增,每个迭代独立计数
3. **问题简述**:
   - 简明扼要(10字以内)
   - 使用中文
   - 反映问题核心

## 🎯 问题类型分类

### 🔷 数学原理类
- 为什么这个公式成立?
- 数学证明是什么?
- 几何意义是什么?

**示例**: `iter01_Q1_向量归一化原理.md`

### 🔶 代码实现类
- 为什么这样写代码?
- 有没有其他实现方式?
- 这个算法的效率如何?

**示例**: `iter05_Q1_叉积为什么用右手定则.md`

### 🔷 应用场景类
- 这个功能在VR中如何应用?
- 实际项目中怎么用?
- 有什么注意事项?

**示例**: `iter10_Q1_矩阵变换在VR中的应用.md`

### 🔶 概念理解类
- 这个概念是什么意思?
- 和其他概念有什么关系?
- 有什么直观的理解方式?

**示例**: `iter15_Q1_齐次坐标的本质.md`

## 💡 使用建议

### 提问时
1. 明确指出哪个迭代、哪个方法
2. 说明具体的疑惑点
3. 可以包含代码片段

### 记录时
1. 使用标准模板
2. 包含完整的解答
3. 添加可视化辅助理解
4. 更新索引文件

### 复习时
1. 按迭代顺序回顾
2. 关注数学原理部分
3. 运行代码验证理解
4. 思考实际应用
